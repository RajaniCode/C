The following samples have been designed to showcase Office development features of Visual Studio 2010.  They can be used to target Office 2007, Office 2010, or later. 

For the best experience please load the versions of the samples corresponding to the version of the .NET Framework you wish to target.  The samples are organized into the following subdirectories:

\Office
    \FX40 - samples optimized to run on the .NET Framework 4.0 Client Profile  [Default for Visual Studio 2010]
    \FX35 - samples optimized to run on the .NET Framework 3.5; they can be modified to work on 4.0 with minor changes described below

For more information about the differences between Office development on 3.5 and 4.0 please see this topic:
Migrating Office Solutions to the .NET Framework 4
